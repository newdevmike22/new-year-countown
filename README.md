# 🎆 New Year Countdown

A dynamic countdown application built with **React** that displays the remaining **days, hours, minutes, and seconds** until the next New Year. The countdown updates in real time and automatically rolls forward each year.

This project was created primarily to gain additional hands-on experience working with **React Hooks**, specifically `useState` and `useEffect`, while building a small but functional application.

## ✨ Features

* Real-time countdown to the next year.
* Displays **days, hours, minutes, and seconds** remaining.
* Updates automatically every second.
* Automatically targets the **next upcoming year**.
* Built using modern **React Hooks**.
* Styled with custom CSS.
* Fast development setup using **Vite**.

## 📂 Project Structure

```
src
│
├── components
│   └── Countdown.jsx
│
├── App.jsx
├── index.css
└── main.jsx
```

### Key Files

**Countdown.jsx**
Contains the core countdown logic and timer display.

**App.jsx**
Main application component that renders the countdown.

**index.css**
Global styles for the application.

**main.jsx**
Application entry point used by Vite to mount the React app.

## ⚙ Installation

Clone the repository and install dependencies.

```bash
git clone https://github.com/your-username/new-year-countdown.git
cd new-year-countdown
npm install
```

Run the development server.

```bash
npm run dev
```

Open the local development server shown in the terminal.
